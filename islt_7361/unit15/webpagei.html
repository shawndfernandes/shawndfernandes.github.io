Image Project

Project Description

The goal of this project is for creating a banner under the topic of parenting. The final image consists of one background image, two foreground images, and text.

Banner 1

The steps taken to create the Final image.

  1. Create a new photo shop document. This can be done by navigating to the File->New option.
  2. Import the background image as a layer. This can be done by navigating to the File-> Place Embedded option, and selecting the background image.
  3. The background image needs to be scaled to the desired size. For this, click on the background image, then navigate to Edit-> Transform -> Scale.
  4. Create a new photoshop document, to open one of the foreground images.
  5. Import the foreground into the new document by navigating to the File-> Place Embedded option, and selecting the foreground image.
  6. In order to Crop the foreground image, the image must be rasterized. After importing the image and clicking on it, photoshop would pop a dialog for converting the smart object into a rasterized image. Select yes.
  7. Once the smart object has been rasterized, select the eraser tool from the left hand side menu.
  8. You may now erase portions of the image, in order to crop out and keep only the required regions of the image. In order to zoom into the image, you can make use of the zoom function beside the toolbar. You may also change the size of the eraser tool by clicking on [ or ] to vary the size. The top of the toolbar below the file menu also gives us the type and shape of eraser tool that can be selected.
  9. One the foreground has been cropped, then drag the foreground to the title of the background image, then a new layer of the foreground image gets created in the background image.
  10. Scale the layer of the foreground image to the required size by navigating to Edit-> Transform -> Scale.
  11. Repeat the same process for the second foreground image, ensuring it gets imported as a new layer over the background image.
  12. We can vary various properties of the layers, such as opacity / transparency, with the tools present in the layer command.
  13. In order to add various style effects to the layer, we can right click the desired layer in the layer toolbox, and then select Blending options. We can then apply various blending options such as Bevel and Emboss, Contour, Texture, Stroke, Inner Shadow, Inner Glow, Satin, Color Overlay, Gradient Overlay, Pattern Overlay, Outer glow and Drop shadow.
  14. The various blending options have numerous settings associated with them, for example, the Gradient Overlay has various options and parameters that can be configured, such as gradient styles and directions of the gradient. I have applied the rainbow gradient overlay to one of my foreground images.
  15. Finally, we need to add text. Click on the text option on the left hand side, and add the required text. If the text is small, then it can be scaled using the Edit-> Transform -> Scale option.
  16. Various text warp options can be applied using the Text Warp button on the top of the page. Different letters can be assigned various sizes and colors, depending on the background, for optimal contrast effect.
  17. We finally export the finished image as a JPG format.

Icon

  1. In order to create the icon, we extract the image from either the cropped foreground or the completed image.
  2. We then place that image in a new photoshop document.
  3. We scale the image to 100 x 100 size, which is ideal for an icon.
  4. To select three colors, we navigate to Image->Mode->Indexed Color.
  5. We then finally export the icon as a JPG image.
