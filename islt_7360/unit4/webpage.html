This is the design document of my final website project. Please provide your valuable feedback

Purpose

The purpose of my final project would be to build a webpage that serves as an online Curriculum Vitae.
The website would accomplish the task of collating all of my social media presence in one place, making it easier to view my profile from single source.
I plan to integrate all of the other URL's into this webpage.

Use cases

  • Highlight prior corporate work experience to prospective employers
  • Highlight prior projects and skills
  • Serve as a single source to all my social media websites, such as LinkedIn, GitHub

Needs / Requirements

It would highlight my projects, internships and work experience in detail.
It would have URL's to various other social media webpages with my profile such as Github, LinkedIn.



Audience

The intended audience for this webpage would be prospective employers.



Questions to Group

Please let me know if any details need to be elaborated


Please provide your valuable feedback and suggestions



Other Reference Websites

LinkedIn, Beyond.com,Indeed.com,Dice.com

Content and structure description

The webpage would consist of

  • Main webpage that hosts the details of the CV
  • Project webpage that hosts details of my projects
  • About me webpage that has a brief description about my educational background and hobbies
  • The header would be fixed and consists of navigation links to various postions of webpage and external websites
  • The footer would also be fixed and contain my email ID for information and feedback.

Color Scheme I plan to make use of high contrast font and background colors for easy reading, and make it more appealing.
I was inspired by the linkedIn and glassdoor color theme.
